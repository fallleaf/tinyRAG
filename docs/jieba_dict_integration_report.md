# jieba 用户自定义词典整合报告

## 整合时间
2026-04-13

## 整合目标
整合 data 目录下的三个 jieba 用户自定义词典文件，统一管理，提高分词准确性。

## 原始词典文件

### 1. jieba_user_dict.txt
- **描述**: 总师支撑词典
- **词条数**: 148 个
- **频率**: 统一为 1000（最高优先级）
- **内容**: 电信网络建设、极简网络、设备退网等专业术语

### 2. jieba_user_dict_16wen_jijian_2026.txt
- **描述**: 党组16号文 + 2026年极简网络专项术语词典
- **词条数**: 42 个
- **频率**: 70-100（中等优先级）
- **内容**: 极简网络专项行动、核心局房整合、架构节点等

### 3. jieba_user_dict_carrier_2026.txt
- **描述**: 运载网相关术语词典
- **词条数**: 0 个（空文件）
- **频率**: 未指定
- **内容**: 无

## 整合策略

1. **去重**: 保留每个词条的唯一性
2. **频率处理**:
   - jieba_user_dict.txt 中的词条频率为 1000（最高优先级）
   - jieba_user_dict_16wen_jijian_2026.txt 中的词条频率为 70-100
   - jieba_user_dict_carrier_2026.txt 中的词条频率设为 100（中等优先级）
3. **优先级**: 同一个词条出现在多个文件时，取最高频率
4. **格式**: 统一为 `词 频率 词性` 格式
